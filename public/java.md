# Définition
Java est un langage de programmation orienté objet qui peut être exécuté sur la plupart des plateformes grâce à la machine virtuelle Java (JVM). Dans un contexte professionnel actuel, c’est l’un des langages les plus utilisés pour le développement Backend et la création d'API qui supportent la logique métier des entreprises. Il reste incontournable côté serveur grâce à des écosystèmes puissants comme Spring (et Spring Boot), qui accélèrent et simplifient la création d'applications web modernes. Cependant, Spring Boot possède un concurrent : Quarkus, un framework principalement basé sur les performances avec un démarrage très rapide ainsi qu’une consommation réduite de ressources.

# Réalisations
J'ai abordé pour la première fois le Java durant ma 3e année avec la mise en place d'une API permettant la création de commandes et leur suivi. Pour ce projet, l’API a été développée en JEE sans l’utilisation de framework. Cela m’a permis de comprendre les différents concepts qui se cachent derrière les frameworks modernes, notamment en configurant les servlets ainsi que les EntityManager à la main.

Par la suite, j’ai consolidé mes acquis et suis monté en compétences en travaillant sur plusieurs projets durant mon alternance. J’ai pu mettre en place un outil permettant l'extraction de données provenant d'un webhook GitLab afin d’automatiser la gestion de scripts SQL. Pour ce projet, j’ai utilisé le framework Spring Boot qui facilite la création d’API. J’ai d’abord commencé par configurer les webhooks sur les projets GitLab. J’ai ensuite récupéré les informations envoyées par les webhooks avec les contrôleurs, pour ensuite les envoyer dans un service afin que les données soient extraites et transformées, avant d'être envoyées au front.
